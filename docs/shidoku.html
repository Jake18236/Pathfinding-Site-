What is Shidoku?

Shidoku is a 4x4 variant of Sudoku, perfect for learning constraint satisfaction problems (CSPs). Fill each row, column, and 2x2 box with the digits 1-4, with no repeats.

Algorithms

  • AC-3 - Arc Consistency algorithm that prunes impossible values from domains before searching.
  • Backtracking + MRV - Depth-first search with Minimum Remaining Values heuristic for variable selection.
  • MAC - Maintaining Arc Consistency runs AC-3 after each assignment for more aggressive pruning.

Controls

Click cells to enter clues (1-4). Use arrow keys to navigate. Press Delete/Backspace to clear a cell.

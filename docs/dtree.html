What is a Decision Tree?

A Decision Tree is a flowchart-like classifier that makes decisions by asking a series of questions about feature values.

How to Use

  • Click anywhere to classify that point
  • Watch the decision path highlight in the Tree tab
  • Adjust depth to control tree complexity
  • Design mode to build your own tree manually

Building the Tree

  1. Start with all samples at root
  2. Find best feature and threshold to split
  3. Create child nodes for each split
  4. Recurse until stopping criteria met

Split Criteria

Gini: G = 1 - Σ(pᵢ)²

Entropy: H = -Σ pᵢ log₂(pᵢ)

Markov Chains

A Markov chain is a stochastic process where the probability of transitioning to any state depends only on the current state (memoryless property).

How to Use
  1. Select a preset: Choose from different example chains
  2. Click Play: Watch the random walk simulation
  3. Observe convergence: See how visit frequencies approach the stationary distribution
Key Concepts
  • Transition Probability: P(next state | current state)
  • Stationary Distribution: Long-run proportion of time in each state
  • Ergodic: Chain that converges to unique stationary distribution

Transition Matrix

The transition matrix P where P[i,j] = probability of going from state i to state j.

Select a preset to see the transition matrix.

Things to Try

Weather Model

Simple 2-state model: Sunny and Rainy days.

  • Watch how quickly it converges to ~67% Sunny, ~33% Rainy
  • Notice the self-loops (staying in same state)
PageRank

4-page web graph simulating random web surfer.

  • Pages with more incoming links get higher visit frequency
  • This is how Google ranks web pages!
Gambler's Ruin

Random walk with absorbing states at $0 and $3.

  • Notice: once you hit $0 or $3, you stay there forever
  • These are "absorbing states"
Drunkard Walk

Symmetric random walk on a line.

  • Equal probability of moving left or right
  • Converges to uniform distribution
